ChatPointsTTV is a Spigot plugin that helps you to create interactions between your stream and your Minecraft world. Set up your own rewards in exchange of channel points, follows, donations, subs and more! Spawn mobs, give items or run any command when an event is triggered.  

There's a whole range of use cases and possibilities to connect with your audience!
